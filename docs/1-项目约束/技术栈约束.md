## 一、开发环境

| 项 | 版本/配置 | 备注 |
|----|-----------|------|
| DevEco Studio | 5.0.0+ (API 12) | 必须使用 API 12 或更高 |
| SDK 版本 | API 12 (HarmonyOS NEXT) | |
| 应用模型 | Stage 模型 | 禁用 FA 模型 |
| 目标设备 | 手机 (Phone) | 模拟器或真机均可 |
| 最低 API 版本 | API 12 | |

---

## 二、强制使用的库

| 功能 | 库名 | 用途 |
|------|------|------|
| 本地数据库 | `@ohos.data.relationalStore` | 核心内容库存储、收藏、历史记录 |
| 网络请求 | `@ohos/axios` | 所有后端接口调用 |
| 状态持久化 | `AppStorage` / `PersistentStorage` | Token 存储、跨页面状态 |
| 页面路由 | `@ohos.router` | pushUrl / replaceUrl 页面跳转 |

---

## 三、禁止使用的库/方式

- ❌ `@ohos.data.preferences`（用户首选项）—— 不能替代 relationalStore 用于核心内容库存储
- ❌ `@ohos.file`（文件存储）—— 结构化数据必须用 RDB
- ❌ `ForEach` 全量渲染列表 —— 必须使用 List + LazyForEach
- ❌ 第三方统计图库（如 ECharts）—— 图表用 Grid 或 Canvas 自绘

---

## 四、API 版本注意

- `relationalStore` 从 API 7 开始支持，API 12 下功能稳定
- `axios` 在 API 20 环境下可正常使用拦截器
- `LazyForEach` 仅在 List、Grid、Swiper、WaterFlow 组件中支持懒加载
